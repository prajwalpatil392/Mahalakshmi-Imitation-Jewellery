📦 Bulk Products JSON Helper

Prepare your product data for bulk import

📋 How to Use:

  1. Paste your product JSON data in the textarea below (or use the sample data)
  2. Click "Validate JSON" to check if the format is correct
  3. Click "Copy to Clipboard" to copy the JSON
  4. Go to Admin Panel → Products → "📦 Bulk Import"
  5. Create a new file (products.json) and paste the copied JSON
  6. Upload the file in the bulk import modal
